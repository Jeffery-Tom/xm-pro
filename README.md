# xm-pro

## 项目简介
这是一个公司部门信息管理项目（简单说一句是做什么的，比如：图书管理系统/学生选课系统/个人博客）

## 技术栈
- Java
- Spring Boot
- MySQL
- MyBatis
- Vue

## 如何运行
1. 克隆项目：`git clone https://github.com/Jeffery-Tom/xm-pro.git`
2. 用IDEA打开
3. 配置数据库连接（在application.yml里修改）
4. 运行 `SpringbootApplication.java`

## 功能介绍
- 功能1：实现员工和管理员的分级的登录与注册
- 功能2：实现管理员对文章、部门和用户管理
- 功能3: 实现员工对个人信息的基本修改和知情公司部门数据统计
- 功能4: 实现对数据库数据的批量导入导出功能,且允许上传图片和个人信息头像等文件
